Modelo de Caixas

O aninhamento em html diz respeito a organização das caixas em formato de ninho.

Height = altura da caixa, Width = largura da caixa, Border = Linha da caixa, Padding = acolchoamento ou preenchimento, Margin = espaço externo à borda, Outline = lado de fora da borda (borda da borda).

Tipos de caixa

Box-level = pula linha ao inserir caixa, Inline-level = cria a caixa na mesma linha.

Exemplos Box-level = div, h1- h6, p, main, header, nav, article, aside, footer, form, video.

Exemplos Inline-level = span, a, code, small, strong, em, sup-sub, label, button, input, select.

User agent = navegador

Podemos utilizar o dev tools do navegador para experimentar os tamanhos das caixas sem alterar o original.

Border style solid = para adicionar borda

Grouping Tags em HTML5

No HTML5 procuramos sempre trabalhar de forma semântica no nosso código. Trabalhar de forma semântica ajuda muito os mecanismos de busca da internet e também em nossa organização.

Header = cabeçalho do site | Main = conteúdo principal do site | Footer = rodapé do site. Todas essas tags são na verdade divs especiais.

Main | section | article | aside

Em style color = cor da letra

Nav = menu de navegação

Para mudar formatações nas tags sempre lembrar se estão aninhadas.

font-weight = mudar formatação da letra (peso).
